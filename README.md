# Shopping Cart

A simple REST application with Kotlin

## Stack

- [Kotlin](https://developer.android.com/kotlin)
- [Javalin](https://javalin.io/)
- [Gradle](https://gradle.org/)

## Building Application

```console
./gradlew clean build
```

## Running Application

```console
java -jar build/libs/shopping-cart-<version>.jar
```
